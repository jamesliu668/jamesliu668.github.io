Sprite Sheet Decomposer

If you are doing gaming programs in Android, IOS or HTML, you will be familior with Sprint Sheet animation. This is a png file (or a series of png files) to represent an animation. This project is written by PHP. It will help you to decompose a png file into several png files.

Google Translate Voice Download

Google Translate is one of the best free online translation services. Besides the translation, it also privdes a very good voice over service. You can get the right pronunciation of the given word. This project will help you to download the pronunciation mp3 from google server.
